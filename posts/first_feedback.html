First feedback

Collected feedback:

Lars Rietbergen: pages are missing content, more use of color, better font style and better use of photos.

Alex Lamper: more color and personality.

Daniël de Jonge: more life in your site.

Daan de Waard: give better names.

While this is the only feedback on paper the general feedback I got from people is that the site is just boring.

Positives:

The positives are almost the same for everyone: the code looks good/clean. While I agree most of this work is done by the plugin Prettier. So not really positive about the work that I have done myself.

Conclusion:

In short: boring site, with clean code. To make the site less boring I had to come up with a better code structure. It may look clean but a lot of elements were not given any type of class and not used properly. So I started over so I could give a better structure for a better workflow. The content remained similar so it is just a matter of copying over to the new structure. A few ideas came to mind as to how to make my site more appealing. I used things like animation and a better color than that ugly green. Changed some content I was not happy with and added some more requirements I did not have before. In the end, I am happy with how I upgraded my site. It is not perfect but it works!
